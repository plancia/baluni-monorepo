# QUICK START

After cloning the repository, install dependencies from the root of the project
```bash
#yarn
yarn install
#npm
npm i
```

### Launch backend
from the root of the project, run the following command:

```bash
yarn backend:dev
```
### Launch web app
from the root of the project, run the following command:

```bash
yarn web:dev
```

### Launch cli (optional)
```bash
yarn cli:dev
```
